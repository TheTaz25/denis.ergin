Wie erstelle ich eine Datenbank in SQL? - Die Lösung

Um eine Datenbank in SQL zu erstellen, nutzt man den Befehl CREATE DATABASE. Hier sind die Schritte zusammengefasst:

  1. Öffne ein SQL-Interface oder eine Datenbankverwaltungssoftware.
  2. Verwende den Befehl: CREATE DATABASE datenbankname;, um eine neue Datenbank zu erstellen.
  3. Wähle die Datenbank aus mit: USE datenbankname;, um darin weiter zu arbeiten.
  4. Erstelle Tabellen mit CREATE TABLE tabellenname (spaltendefinitionen);.
