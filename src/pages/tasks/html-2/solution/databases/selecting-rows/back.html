Wie werden Daten aus einer Datenbank abgefragt? - Die Lösung

Um Daten aus einer SQL-Datenbank abzufragen, verwendet man den Befehl SELECT. Die grundlegenden Schritte sind:

  1. Nutze SELECT spaltenname oder SELECT *, um alle Spalten auszuwählen.
  2. Gib mit FROM tabellenname die Tabelle an, aus der du die Daten abfragen möchtest.
  3. Verwende optional WHERE, um die Abfrage auf bestimmte Kriterien zu beschränken.
  4. Nutze ORDER BY, um die Ergebnisse zu sortieren.
  5. Beispiel: SELECT * FROM kunden WHERE stadt = 'Berlin'; fragt alle Kunden aus Berlin ab.
